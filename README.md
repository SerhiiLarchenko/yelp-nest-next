## Description

Yelp, NestJS, React project demo
## Installation

## Requirements
### Requirements

1. [Node.js](https://nodejs.org/en/) >= v12.16.3
2. [yarn](https://yarnpkg.com/) >= v1.21.1
### Back-end

```bash
$ yarn
```
### Front-end

```bash
$ cd front-end
$ yarn
```
## Running the app

### Back-end

```bash
$ yarn start
```
### Front-end

```bash
$ cd front-end
$ yarn start
```
Open [read more here](http://localhost:3001) in your browser to check out the app
